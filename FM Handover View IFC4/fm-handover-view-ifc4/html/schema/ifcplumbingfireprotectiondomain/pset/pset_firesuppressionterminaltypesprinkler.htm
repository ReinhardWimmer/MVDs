7.6.4.5 Pset_FireSuppressionTerminalTypeSprinkler

PSET_TYPEDRIVENOVERRIDE / IfcFireSuppressionTerminal / SPRINKLER

Properties

NameTypeDescription
SprinklerType P_ENUMERATEDVALUE / IfcLabel / PEnum_SprinklerType Identifies the predefined types of sprinkler from which the type required may be set.
Activation P_ENUMERATEDVALUE / IfcLabel / PEnum_SprinklerActivation Identifies the predefined methods of sprinkler activation from which that required may be set.
Response P_ENUMERATEDVALUE / IfcLabel / PEnum_SprinklerResponse Identifies the predefined methods of sprinkler response from which that required may be set.
ActivationTemperature P_SINGLEVALUE / IfcThermodynamicTemperatureMeasure The temperature at which the object is designed to activate.
CoverageArea P_SINGLEVALUE / IfcAreaMeasure The area that the sprinkler is designed to protect.
HasDeflector P_SINGLEVALUE / IfcBoolean Indication of whether the sprinkler has a deflector (baffle) fitted to diffuse the discharge on activation (= TRUE) or not (= FALSE).
BulbLiquidColor P_ENUMERATEDVALUE / IfcLabel / PEnum_SprinklerBulbLiquidColor The color of the liquid in the bulb for a bulb activated sprinkler. Note that the liquid color varies according to the activation temperature requirement of the sprinkler head. Note also that this property does not need to be asserted for quick response activated sprinklers.
DischargeFlowRate P_SINGLEVALUE / IfcVolumetricFlowRateMeasure The volumetric rate of fluid discharge.
ResidualFlowingPressure P_SINGLEVALUE / IfcPressureMeasure The residual flowing pressure in the pipeline at which the discharge flow rate is determined.
DischargeCoefficient P_SINGLEVALUE / IfcReal The coefficient of flow at the sprinkler.
MaximumWorkingPressure P_SINGLEVALUE / IfcPressureMeasure Maximum pressure that the object is manufactured to withstand.
ConnectionSize P_SINGLEVALUE / IfcPositiveLengthMeasure Size of the inlet connection to the sprinkler.
