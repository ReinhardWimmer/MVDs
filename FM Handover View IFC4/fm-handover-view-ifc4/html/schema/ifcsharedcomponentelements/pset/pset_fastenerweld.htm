6.3.4.10 Pset_FastenerWeld

PSET_TYPEDRIVENOVERRIDE / IfcFastener / WELD

Properties

NameTypeDescription
Type1 P_SINGLEVALUE / IfcLabel Type of weld seam according to ISO 2553. Note, combined welds are given by two corresponding symbols in the direction of the normal axis of the coordinate system. For example, an X weld is specified by Type1 = 'V' and Type2 = 'V'.
Type2 P_SINGLEVALUE / IfcLabel See Type1.
Surface1 P_SINGLEVALUE / IfcLabel Aspect of weld seam surface, i.e. 'plane', 'curved' or 'hollow'. Combined welds are given by two corresponding symbols analogous to Type1 and Type2.
Surface2 P_SINGLEVALUE / IfcLabel See Surface1.
Process P_SINGLEVALUE / IfcInteger Reference number of the welding process according to ISO 4063, an up to three digits long code
ProcessName P_SINGLEVALUE / IfcLabel Name of the welding process. Alternative to the numeric Process property.
a P_SINGLEVALUE / IfcPositiveLengthMeasure Measure a according to ISO 2553

l P_SINGLEVALUE / IfcPositiveLengthMeasure Measure l according to ISO 2553
n P_SINGLEVALUE / IfcCountMeasure Count n according to ISO 2553
s P_SINGLEVALUE / IfcPositiveLengthMeasure Measure s according to ISO 2553
z P_SINGLEVALUE / IfcPositiveLengthMeasure Measure z according to ISO 2553
Intermittent P_SINGLEVALUE / IfcBoolean If fillet weld, intermittent or not
Staggered P_SINGLEVALUE / IfcBoolean If intermittent weld, staggered or not
