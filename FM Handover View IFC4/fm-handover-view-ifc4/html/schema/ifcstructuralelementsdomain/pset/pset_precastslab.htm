7.8.4.6 Pset_PrecastSlab

PSET_TYPEDRIVENOVERRIDE / IfcSlab

Properties

NameTypeDescription
TypeDesignator P_SINGLEVALUE / IfcLabel Type designator for the precast concrete slab, expressing mainly the component type. Possible values are “Hollow-core”, “Double-tee”, “Flat plank”, etc.
ToppingType P_SINGLEVALUE / IfcLabel Defines if a topping is applied and what kind. Values are “Full topping”, “Perimeter Wash”, “None”
EdgeDistanceToFirstAxis P_SINGLEVALUE / IfcPositiveLengthMeasure The distance from the left (‘West’) edge of the slab (in the direction of span of the components) to the axis of the first component.
DistanceBetweenComponentAxes P_SINGLEVALUE / IfcPositiveLengthMeasure The distance between the axes of the components, measured along the ‘South’ edge of the slab.
AngleToFirstAxis P_SINGLEVALUE / IfcPlaneAngleMeasure The angle of rotation of the axis of the first component relative to the ‘West’ edge of the slab.
AngleBetweenComponentAxes P_SINGLEVALUE / IfcPlaneAngleMeasure The angle between the axes of each pair of components.
NominalThickness P_SINGLEVALUE / IfcPositiveLengthMeasure The nominal overall thickness of the slab.
NominalToppingThickness P_SINGLEVALUE / IfcPositiveLengthMeasure The nominal thickness of the topping.
