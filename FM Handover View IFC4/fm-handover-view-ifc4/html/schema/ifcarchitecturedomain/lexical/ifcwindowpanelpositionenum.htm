Semantic definitions at the type

Type definition

This enumeration defines the basic configuration of the window type in terms of the location of window panels. The window configurations are given for windows with one, two or three panels (including fixed panels) as shown in Figure 115. It corresponds to the OperationType of the IfcWindowStyle definition, which references the IfcWindowPanelProperties.

Windows which are subdivided into more than three panels have to be defined by the geometry only. The type of such windows is given by an IfcWindowType.OperationType = USERDEFINED or NOTDEFINED (see IfcWindowStyleOperationEnum for details).
