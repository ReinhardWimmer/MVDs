7.4.4.3 Pset_AudioVisualApplianceTypeCamera

PSET_TYPEDRIVENOVERRIDE / IfcAudioVisualAppliance / CAMERA

Properties

NameTypeDescription
CameraType P_ENUMERATEDVALUE / IfcLabel / PEnum_AudioVisualCameraType Indicates the type of camera.
IsOutdoors P_SINGLEVALUE / IfcBoolean Indicates if camera is designed to be used outdoors.
VideoResolutionWidth P_SINGLEVALUE / IfcInteger Indicates the number of horizontal pixels (the largest native video resolution width).
VideoResolutionHeight P_SINGLEVALUE / IfcInteger Indicates the number of vertical pixels (the largest native video resolution height).
VideoResolutionMode P_TABLEVALUE / IfcIdentifier / IfcLabel Indicates video resolution modes.
VideoCaptureInterval P_TABLEVALUE / IfcIdentifier / IfcTimeMeasure Indicates video frame capture time intervals.
PanTiltZoomPreset P_TABLEVALUE / IfcIdentifier / IfcLabel Indicates pan/tilt/zoom position presets.
PanHorizontal P_BOUNDEDVALUE / IfcLengthMeasure Indicates horizontal range for panning.
PanVertical P_BOUNDEDVALUE / IfcLengthMeasure Indicates vertical range for panning.
TiltHorizontal P_BOUNDEDVALUE / IfcPlaneAngleMeasure Indicates horizontal range for pivoting, where positive values indicate the camera rotating clockwise,
TiltVertical P_BOUNDEDVALUE / IfcPlaneAngleMeasure Indicates vertical range for pivoting, where 0.0 is level, +90 degrees is looking up, -90 degrees is looking down.
Zoom P_BOUNDEDVALUE / IfcPositiveLengthMeasure Indicates the zoom range.
