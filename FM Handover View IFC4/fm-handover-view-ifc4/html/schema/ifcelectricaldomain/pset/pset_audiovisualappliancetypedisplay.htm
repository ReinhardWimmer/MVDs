7.4.4.5 Pset_AudioVisualApplianceTypeDisplay

PSET_TYPEDRIVENOVERRIDE / IfcAudioVisualAppliance / DISPLAY

Properties

NameTypeDescription
DisplayType P_ENUMERATEDVALUE / IfcLabel / PEnum_AudioVisualDisplayType Indicates the type of display.
NominalSize P_SINGLEVALUE / IfcPositiveLengthMeasure Indicates the diagonal screen size.
DisplayWidth P_SINGLEVALUE / IfcPositiveLengthMeasure Indicates the physical width of the screen (only the display surface).
DisplayHeight P_SINGLEVALUE / IfcPositiveLengthMeasure Indicates the physical height of the screen (only the display surface).
Brightness P_SINGLEVALUE / IfcIlluminanceMeasure Indicates the display brightness.
ContrastRatio P_SINGLEVALUE / IfcPositiveRatioMeasure Indicates the display contrast ratio.
RefreshRate P_SINGLEVALUE / IfcFrequencyMeasure Indicates the display refresh frequency.
TouchScreen P_ENUMERATEDVALUE / IfcLabel / PEnum_AudioVisualDisplayTouchScreen Indicates touchscreen support.
VideoResolutionWidth P_SINGLEVALUE / IfcInteger Indicates the number of horizontal pixels, e.g. 1920.
VideoResolutionHeight P_SINGLEVALUE / IfcInteger Indicates the number of vertical pixels, e.g. 1080.
VideoResolutionMode P_TABLEVALUE / IfcIdentifier / IfcLabel Indicates video resolution modes.
VideoScaleMode P_TABLEVALUE / IfcIdentifier / IfcLabel Indicates video scaling modes.
VideoCaptionMode P_TABLEVALUE / IfcIdentifier / IfcLabel Indicates video closed captioning modes.
AudioMode P_TABLEVALUE / IfcIdentifier / IfcLabel Indicates audio sound modes and corresponding labels, if applicable.
