Semantic definitions at the entity

Entity definition

The IfcReparametrisedCompositeCurveSegment is geometrically identical to a IfcCompositeCurveSegment but with the additional capability of reparametrization.

NOTE  Definition according to ISO/CD 10303-42:1992
The reparametrised composite curve segment is a special type of composite curve segment which provides the capability to re-define its parametric length without changing its geometry.

Let l = ParamLength.
