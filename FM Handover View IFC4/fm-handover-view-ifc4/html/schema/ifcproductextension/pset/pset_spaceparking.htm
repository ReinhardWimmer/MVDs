5.4.4.22 Pset_SpaceParking

PSET_TYPEDRIVENOVERRIDE / IfcSpace / PARKING

Properties

NameTypeDescription
ParkingUse P_SINGLEVALUE / IfcLabel Identifies the type of transporation for which the parking space is designed. Values are not predefined but might include car, compact car, motorcycle, bicycle, truck, bus etc.
ParkingUnits P_SINGLEVALUE / IfcCountMeasure Indicates the number of transporation units of the type specified by the property ParkingUse that may be accommodated within the space. Generally, this value should default to 1 unit. However, where the parking space is for motorcycles or bicycles, provision may be made for more than one unit in the space.
IsAisle P_SINGLEVALUE / IfcBoolean Indicates that this parking zone is for accessing the parking units, i.e. an aisle (TRUE) and not a parking unit itself (FALSE)
IsOneWay P_SINGLEVALUE / IfcBoolean Indicates whether the parking aisle is designed for oneway traffic (TRUE) or twoway traffic (FALSE). Should only be provided if the property IsAisle is set to TRUE.
