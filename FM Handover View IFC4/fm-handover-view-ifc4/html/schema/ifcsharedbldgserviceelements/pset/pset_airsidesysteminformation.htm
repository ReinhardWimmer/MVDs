Properties

NameTypeDescription
Name P_SINGLEVALUE / IfcLabel The name of the air side system.
Description P_SINGLEVALUE / IfcLabel The description of the air side system.
AirSideSystemType P_ENUMERATEDVALUE / IfcLabel / PEnum_AirSideSystemType This enumeration specifies the basic types of possible air side systems (e.g., Constant Volume, Variable Volume, etc.).

EnergyLoss P_SINGLEVALUE / IfcPowerMeasure The sum of energy losses for the spaces served by the system during the peak heating conditions.
