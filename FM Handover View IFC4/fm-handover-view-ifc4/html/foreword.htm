Foreword

Update of the IFC2x3 based FM Handover View to IFC4 Add2.

First Draft

General Information

New defined Concepts and Properties, according to FM Handover View.


Concepts can be found under Operation Concepts


Property Sets have the Prefix POpe for Properties for Operation.

Questions and Comments

FMHOV uses Coordination View as a reference view. This could be handled by RV or DTV for IFC 4 if necessary. Needs to be checked!


How shall we handle the entity Furniture, Fixture and Equipment ? It is defined as Any other furniture, fixture and equipment


Space Boundaries are defined seperately for Door and Window. This should be covered by the 1st level Space Boundary concept, assigned to IfcSpace. Needs to be checked!


Properties for IfcSpace require a Space Cladding. This Property is not defined and is covered by other Properties, like wall covering, ...


Building Service Element MEP is represented by IfcDistributionFlowElement as an abstract entity of all HVAC components


Project Representation Context is not exactly what the FMHOV demands. Needs to be checked.


ERM export requirements do not comply with the concepts, parts of concepts are set to be mandatory for the export, while some parts are not. Decision: If the some parts of concepts are meant to be mandatory for the export, the export of the concept will be set to mandatory.
